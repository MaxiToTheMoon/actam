# bordero-client

This template should help get you started developing with Vue 3 in Vite.

## Executing the frontend part of the project

To run the project, first run the python main.py file in the backend

In another cmd instance you have install the required packages and dependencies, to do that navigate to "frontend\bordero-client" and run "npm install", to test the frontend execute the command "npm run dev". 
It has been written using the Vue 3 framework with Vuetify.
